Comment pouvons-nous vous aider ?

Naviguer dans le webdocumentaire :

Sur la page d’accueil il y a une carte interactive de Paris qui vous permet d’atteindre les différentes stations de métro présentées dans ce webdocumentaire. Sur la carte, pour atteindre une station, vous devez cliquer sur le lien portant le nom de la station.

La barre de navigation principale de ce webdocumentaire est composée des éléments suivant : une rubrique “À propos”, une rubrique “Crédits”, une rubrique “Aide” et une liste déroulante permettant de changer la langue des contenus présentés dans le cadre de ce webdocumentaire.

Toutes les informations sur ce projet de webdocumentaire sont accessibles dans la rubrique “À propos”.

Vous avez accès à toutes les informations sur l’équipe et les sources qui ont été utilisées pour réaliser ce webdocumentaire dans la rubrique “Crédits”.

Vous avez également accès à une rubrique “Aide”, dans la barre de navigation principale.

Pour changer la langue des contenus de ce webdocumentaire vous pouvez utiliser le menu déroulant situé dans la barre de navigation principale.

Une fois que vous avez quitté la page d’accueil et êtes donc dans l’une des stations de métro présentées par le webdocumentaire, Pour visiter une autre station vous pouvez utiliser la barre de navigations secondaire de ce webdocumentaire. Concrètement c’est une une ligne de métro reliant les différentes stations. Ces différentes stations sont cliquables et vous redirigent vers la station voulue. De plus, vous pouvez utiliser les flèches qui encadrent la ligne de métro et qui vous redirigent vers la station précédent ou suivante.

Quand vous vous trouvez dans l’une des stations de métro de ce webdocumentaire :

Le bouton “Accueil” vous permet de revenir à l'accueil et donc à la carte interactive de Paris.

Dans chaque station de métro il y a cinq éléments cliquables (cinq illustrations) qui représentent des anecdotes tirées du documentaire Métronome de Lorànt Deutsch. Au clic sur l’un de ces éléments, l’extrait vidéo traitant de l’anecdote sélectionné se lance et vous avez également accès à un court résumé de ce que vous allez pouvoir trouver dans cet extrait.

Dans chaque station de métro vous avez donc accès à cinq extraits différents du documentaire Métronome traitant d’autant d’anecdotes. Ces extraits sont rendus accessibles par le clic sur différents éléments composant la station de métro en question. Cela peut être une banderole accrochée au plafond, une béquille posée contre le mur, une affiche, un graffiti ou encore un personnage posté sur le quai.
